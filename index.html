Máquina de Turing

    • Velocidad:

    transformador_aaa

    Detalles:

    transformador_aaa de cadenas. Esta máquina transforma una cadena de símbolos a, b o c y queda a,a,a.

    Ejemplos:

    aaabbbccc
    acaaaacab
    abacbacba

    paso 1: Dar clip en transformador_aaa
    paso2: ingresar una nueva cadena
    paso3: cargar la cadena
    paso4: evaluar la cadena
    paso5: para ingresar una nueva cadena borre la anterior y carge de nuevo
    nota para aumentar la Velocidad primero debe cagar la cadena despues la Velocidad y evaluar

    Palíndromos

    Detalles:

    Verificador de palíndromo. Esta máquina verifica si una cadena es un palíndromo. Si la cinta queda vacía, la cadena es un palíndromo.

    Ejemplos:

    aaabbbaaa
    ccaaaacc
    cbacbcabc
